# News EDA & Topic Analysis (Task 1)

This repository contains code to perform exploratory data analysis (EDA), NLP topic modeling, and time-series analysis on news article data.

## Project Structure

- `.vscode/`: VSCode-specific settings
- `.github/workflows/`: CI/CD workflows
- `src/`: Python modules
- `notebooks/`: Jupyter notebooks for EDA and modeling
- `tests/`: Unit tests
- `scripts/`: Helper scripts
- `requirements.txt`: Python dependencies
